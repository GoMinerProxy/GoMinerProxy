GoMinerProxy 是什么？

GoMinerProxy 是一款基于 GoLang 的高性能多線程 ETH 礦池轉發中繼代理工具，穩定安全，支持SSL、自定義抽水、熱修改、獨家抽水算法、最完美的Web管理、完善社區支持。BTC/ETC/LTC等其他幣種將在后續的更新中增加支持。

問題反饋

如果在使用過程中發現了什么缺陷，或是有新的需求提議，請先檢查一下以前的文檔、issue 是否提過。

如果疑似 Bug 或是功能提議，請創建一個 issue 用于追蹤問題；

如果是日常使用上的疑問，請到 Telegram 討論群組 內詳細描述你遇到的問題。

聯系

你可以通過以下方式與其他正在使用 GoMinerProxy 的用戶或開發者交流：

  • Telegram 討論群組
  • Telegram 通知頻道
  • GitHub
  • GitHub Issue
